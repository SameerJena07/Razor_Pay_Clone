# 💳 Razorpay Clone

A **Razorpay Landing Page Clone** built using **HTML, CSS, JavaScript, and Tailwind CSS**.
This project replicates the design of the Razorpay website to practice **frontend development, responsive layouts, and Tailwind CSS utility classes**.

---

## 🚀 Features

* Modern Razorpay-style landing page
* Fully responsive design
* Clean UI with Tailwind CSS
* Navigation bar with sections
* Hero section with call-to-action
* Feature sections
* Footer layout similar to Razorpay
* Mobile-friendly layout

---

## 🛠️ Technologies Used

* **HTML5** – Page structure
* **CSS3** – Additional styling
* **JavaScript (Vanilla JS)** – Interactive behavior
* **Tailwind CSS** – Utility-first CSS framework for fast styling

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/SameerJena07/Razor_Pay_Clone.git
```

### 2️⃣ Open the project folder

```bash
cd Razor_Pay_Clone
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Run the project

```bash
npm run start
```

Then open the project in your browser.
